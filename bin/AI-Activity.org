#+title: AI Activity Guidance --- custom (Per-Project Template)

This file was installed as the starting template for the =custom=
activity. The =custom= activity is the escape hatch for a project whose
conventions are not covered by any of the standard activities (=basic=,
=bisos-pip=, =xu-single=, =blee-panels=, =lcnt=, etc.) --- rather than
symlinking a shared =AI-Activity.org= that would apply globally, this
activity assumes each project owns its own copy and edits it freely.

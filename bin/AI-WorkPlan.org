*** markitdown

Yes, familiar. markitdown (microsoft/markitdown) is a Python library and CLI tool that
converts PDF, Word, Excel, PowerPoint, HTML, images, audio, ZIP archives, etc. to Markdown.
Installed via pip (not pipx) into =/bisos/venv/py3/asc= so it is both importable as a
library (=from markitdown import MarkItDown=) and available as a CLI command.

*** pkgsSeed pattern for pip installs

=pkgsSeed= has =pipPkg= (=pp=) as a first-class peer to =aptPkg= (=ap=). No custom function
is needed for a plain pip install. The clean pattern:

#+begin_src python
pp = pkgsSeed.pipPkg
pkgsSeed.pkgsSeedInfo.virtenv = "/bisos/venv/py3/asc"

pipPkgsList = [
    pp("markitdown"),
]
pkgsSeed.setup(pipPkgsList=pipPkgsList)
#+end_src

The =func== pattern is only for cases requiring custom shell logic (e.g. adding a new apt
source). For ordinary pip installs, =pp()= is sufficient.

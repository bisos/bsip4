* Project Overview
This repo (=bisos/bsip4=) is part of the BISOS platform layer. It provides
provisioning scripts, SBOM (Software Bill of Materials) command services, and
systemd unit seeds for various capabilities deployed on BISOS/Debian systems.

** Design Decisions
- SQLite + SequentialExecutor (Airflow defaults) are used — appropriate for
  testing a small number of DAGs; no external DB required.
- =assemble=None= in the CBS for now — the assemble step is where
  =airflow db migrate= and initial admin user creation will eventually go.
- The three sysd unit seeds use =pathPlus.whichBinPath("airflow")= to locate
  the executable dynamically from the active venv.

** What Is Untested
- The three =*-sysd.cs= files and the CBS (=airflow-cbs-is-np-sysd.cs=) have
  not been exercised via =cba-sysd.cs= yet — systemd registration is untested.
- No DAGs have been written or tested yet.
- PostgreSQL/MySQL backend path is not set up.

** Next Steps (when resuming)

- Write and test a minimal DAG under =$AIRFLOW_HOME/dags/=.
- Exercise the CBS: run =airflow-cbs-is-np-sysd.cs= to register and start
  the three systemd services.
